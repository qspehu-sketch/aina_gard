# Айна Гард — AI Creator (лендинг)

Статический одностраничный сайт: неон / glass, flip-карточки, фоновая музыка по клику (`body.music-on`), RU/EN, адаптивная вёрстка.

## Структура

```
/assets
  /images   — favicon и изображения
  /audio    — положите сюда ambient.mp3 (опционально; иначе подхватится запасной источник из разметки)
/css
  styles.css
/js
  main.js
index.html
```

## Локальный запуск

Откройте `index.html` в браузере или поднимите любой статический сервер в корне проекта.

## GitHub Pages

1. Создайте репозиторий и загрузите файлы (ветка `main` / `master`).
2. **Settings → Pages → Build and deployment → Source:** Deploy from a branch.
3. Выберите ветку и папку **/ (root)**.
4. Сайт будет доступен по адресу `https://<user>.github.io/<repo>/`.

Пути к `css/`, `js/`, `assets/` относительные — подойдут для проектного Pages. Для пользовательского домена настройте DNS и при необходимости добавьте `CNAME` в репозиторий.

## Музыка

Добавьте файл `assets/audio/ambient.mp3`. Первый `<source>` в `index.html` указывает на него; если файла нет, браузер возьмёт следующий источник (fallback).
